# Flask App for Palm Oil Log

This is a simple Flask application that serves the index.html template.

## Setup

1. Ensure Python is installed.
2. Install dependencies: `pip install -r requirements.txt`

## Running the App

Run the application with:

```
python app.py
```

Then, open your browser and go to `http://127.0.0.1:5000/`

## Troubleshooting

- If port 5000 is in use, Flask will use a different port. Check the terminal output.
- Ensure the templates folder contains index.html.
